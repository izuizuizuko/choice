## :point_up:実装内容
#####  フロントエンド
* Haml/Sassでのマークアップ
* Bootstrapでのマークアップ

#####  サーバーサイド
* ユーザー登録(/編集)機能(gem Deviseを使用)
* 管理機能(gem Activeadminを使用)
* 検索機能(gem ransackを使用)

## 開発環境
* Ruby 2.5.1
* Ruby on Rails 5.2.4.1
* MySQL 5.6.43
* Haml 
* Sass 
* AWS
* EC2
* Github
